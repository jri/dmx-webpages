Hello,

this the default frontpage of the dm4-webpages module.

The curent development here is that there is not yet an index of all your pages, except that one in JSON.

This is mainly due to keep up the possibility for other plugins to register their own frontpages on the dm4-webpages plugin through calling for example:

WebpagePluginService.setFrontpageResource('/web/index.html')

Nonetheless, all the pages you already published are available via their web alias, e.g. http://www.domain.xyz/page/my-web-alias.
